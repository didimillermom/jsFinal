104 The Ranch

Web design and development

This website was created for Cache Valley Media Group. The instructions were to create a single page with a large header at the top with three classic country artists. They wanted to keep all of the sites fluent and consistent in design. The website was created using Wordpress and the Divi theme. The graphics were modified by myself but the original images are all Creative Commons. Later additional pages were added to keep up with the contests and news.
